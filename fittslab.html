Über FittsLab

FittsLab ist ein experimentelles Tool zur Messung und Analyse der Benutzerfreundlichkeit von Webinterfaces basierend auf dem Fitts'schen Gesetz. Dieses fundamentale Prinzip der Mensch-Computer-Interaktion beschreibt die Zeit, die benötigt wird, um ein Ziel in Abhängigkeit von Distanz und Größe zu erreichen.

Die Fitts'sche Formel:

MT = a + b × log₂(2D/W)

Verwendete Formulierung: Fitts (1954). Alternative: Shannon-Formulierung log₂(D/W + 1).

  • MT = Movement Time (Bewegungszeit in ms)
  • a = Intercept (Startzeit/Reaktionszeit)
  • b = Slope (Bewegungsgeschwindigkeit)
  • D = Distance (Distanz zum Ziel)
  • W = Width (Breite des Ziels)
  • ID = Index of Difficulty = log₂(2D/W)

Funktionen:

  • Präzise Zeitmessung mit performance.now() API (±5ms Genauigkeit)
  • Hit/Miss-Tracking für Genauigkeitsanalyse
  • Vergleich zwischen theoretischen und gemessenen Werten
  • Regression und R²-Berechnung
  • Export aller Messdaten als CSV
  • Analyse echter HTML-Elemente (Buttons, Links, Inputs)
Forschungskontext:
Entwickelt im Rahmen der Bachelorarbeit zur Anwendung des Fitts'schen Gesetzes in webbasierten Anwendungen zur messbaren Bewertung der Benutzerfreundlichkeit.
